API

На сайте доступно публичное HTTP API в формате JSON.

Информация о списке фильмов и их детальной инормации доступна по адресу /api/get_films_list.

Для уточнения информации в запросе доступны ключи:
  • top_size - Размер итогового списка (с сортировкой по рейтингу фильма).
  • cinemas_over - Минимальное количество кинотеатров, в которых идет показ фильма.
  • rating_over - Минимальный рейтинг фильма.
  • city - Фильтр фильмов по городу.

Информация о конкретном фильме доступна по адресу: /api/movie/<film_id>.

В этом запросе доступен единственный ключ: city для получения информации о количестве кинотеатров в которых идет показ фильма в указанном городе.
